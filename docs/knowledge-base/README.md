# Knowledge Base — VLA Research Intelligence (snapshot 2026-05)

[← Back to repo README](../../README.md)

This knowledge base preserves the **full research record** gathered while producing the 8 proposals — far more than the proposals themselves contain. It was compiled by ~20 parallel research/red-team agents over the VLA literature (2023–2026) and is captured here so the intelligence isn't lost.

> ⚠️ **Snapshot caveat.** Reflects the literature as of **May 2026**, a field moving ~1 paper/subarea/3 weeks. Treat every arXiv ID as *to-be-re-verified*; a handful of very recent IDs came from automated search and need a human check before citing.

## Contents
1. **[VLA Landscape](01-vla-landscape.md)** — model catalog 2023–2026, convergence map, what's saturated, the open gaps.
2. **[Open Problems & Evaluation](02-open-problems-and-evaluation.md)** — ranked open problems with severity/citations, the benchmark catalog and its known flaws, the reproducibility crisis.
3. **[Publication Strategy](03-publication-strategy.md)** — venue comparison (conf vs journal), accept/reject checklists, recent accepted hooks, paper-type ROI for an academic lab.
4. **[Technique Survey](04-technique-survey.md)** — the big cross-domain win-win scorecard: backbones, RL/reasoning, generative heads, representations/world-models, systems efficiency, wildcards, and cross-disciplinary imports (control/info-theory/neuro).
5. **[Red-Team Analyses](05-red-team-analyses.md)** — the detailed adversarial reviews that killed the leading method ideas (objections + scooping papers + salvage angles).

See also: the [methodology](../methodology.md) (how this was produced), the [research log](../research-log.md) (the kill summary), and the [bibliography](../bibliography.md).
